Overview

TMDB and Chill is a Flask-powered API data visualization project that provides an overview of movie genres from 1916 to 2016. The data for this project was obtained from Kaggle and plotted in a visual manner, allowing users to easily compare and analyze the trends and patterns of the movie industry over the years. This project aims to provide valuable insights into the evolution of movie genres and the factors that influence their success, as well as to serve as a useful resource for researchers, movie enthusiasts, and industry professionals alike.

Tools & Technology

Below are some of the tools and technologies used to create the visualizations:

  • Python
  • Flask
  • SQlite
  • JavaScript
  • Node.js
  • D3
  • CSS
  • HTML

Biases & Further Considerations

Some factors that we should consider when analyzing this dataset are as follows:

  • Data quality analysis has not been performed
    • Some genres were provided with a '#N/A' if genre was unknown
    • Missing budget and/or revenue were provided a value of zero
    • Revenue currency is not provided
  • Selection bias:
    • Dataset is not a full depiction of all movies (i.e., independent films, foreign films)
    • Self-imposed budget and revenue parameters for filtering
